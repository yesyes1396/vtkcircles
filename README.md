# Система записи на кружки и секции

Веб-приложение для управления регистрацией пользователей в учебные секции с административной панелью.


## Основной функционал
* **Каталог:** Просмотр секций с фильтрацией по категориям и возрасту.
* **Авторизация:** Система регистрации и входа для пользователей.
* **Личный кабинет:** Управление активными записями.
* **Админ-панель:** Управление списком курсов и просмотр студентов.
* **Валидация:** Контроль лимита мест и возрастных ограничений.

## Технологический стек
* **Backend:** Python 3.10, Flask, SQLAlchemy.
* **Frontend:** Jinja2, Bootstrap 5.
* **Инфраструктура:** Docker, Docker Compose, SQLite.

## Модели данных
| Модель | Поля |
| --- | --- |
| **User** | id, username, email, password_hash, age, is_admin |
| **Course** | id, name, category, min_age, max_students, schedule |
| **Enrollment** | user_id, course_id, enrollment_date |
