Система записи на кружки и секции
Веб-приложение на Flask для управления записью в учебные секции с административной панелью.

## Основные возможности
Каталог: Поиск и фильтрация кружков по категориям и возрасту.

Личный кабинет: Управление своими записями и профилем.

Админка: Управление списком курсов и просмотр списков учеников.

Контроль мест: Автоматическая блокировка записи при заполнении группы.

Адаптивность: Корректная работа на мобильных устройствах.

Технологии
Backend: Python 3.10+, Flask, SQLAlchemy (ORM).

Frontend: Bootstrap 5, Jinja2.

База данных: SQLite (по умолчанию).

Развертывание: Docker, Docker Compose.

# Запуск проекта
## Вариант 1: Через Docker (рекомендуется)
docker-compose up -d
Приложение будет доступно по адресу: http://localhost:5000

## Вариант 2: Локально (Python)
Установите зависимости: pip install -r requirements.txt

## нициализируйте базу данных:

python -c "from app.app import db; db.create_all()"
python app/app.py seed-db
Запустите сервер: python app/app.py runserver
