📋 ОСНОВНЫЕ МАРШРУТЫ
═════════════════════════════════════════════════════════════════════════════

ПУБЛИЧНЫЕ:
  GET  /                    - Главная страница (каталог)
  GET  /course/<id>         - Детали кружка
  POST /enroll/<id>         - Запись на кружок
  POST /unenroll/<id>       - Отписка от кружка

АВТОРИЗАЦИЯ:
  GET  /register            - Форма регистрации
  POST /register            - Обработка регистрации
  GET  /login               - Форма входа
  POST /login               - Обработка входа
  GET  /logout              - Выход из системы

ПОЛЬЗОВАТЕЛЬ (требует auth):
  GET  /profile             - Личный кабинет

АДМИНИСТРАТОР (требует admin):
  GET  /admin               - Главная админ-панель
  GET  /admin/course/add    - Форма добавления
  POST /admin/course/add    - Добавление кружка
  GET  /admin/course/<id>/edit   - Форма редактирования
  POST /admin/course/<id>/edit   - Редактирование
  POST /admin/course/<id>/delete - Удаление
  GET  /admin/course/<id>/enrollments - Список записавшихся
